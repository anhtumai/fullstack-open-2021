## Installation

```bash
$> git clone git@github.com:anhtumai/blog-list-frontend.git
$> cd blog-list-frontend
$> npm install
```


### Run

```bash
$> npm start
```

### Test

#### 1. React Testing Library

```bash
// use yarn
$> yarn test
// or use npm
$> npm run test
```

#### 2. Cypress

```bash
$> npm run cypress:open
```
