# Ultimate Hook

> Frontend for Ultimate Hook project using React-Hook.
> Assignment for course Fullstack open 2021.
> Written in ReactJs 

[Course link](https://fullstackopen.com/en/part7/)

## Demo

![Gif](demo/ultimate-hook.gif)

## Run

Install dependencies:
```bash
$> npm install
```

Run Json server:
```bash
$> npm run server
```

Run project:
```bash
$> npm start
...
```
